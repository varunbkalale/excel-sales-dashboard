# 📊 Sales Dashboard (Excel)

## 📌 Overview
This project presents a sales dashboard built in Excel to analyze revenue, profit and performance across regions and products.

---

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts

---

## 📁 Project Structure
- sales_dashboard.xlsx → Excel dashboard  
- screenshots/ → dashboard preview  

---

## 📊 Dashboard Preview

![Dashboard](screenshots/dashboard_overview.png)

---

## 🔍 Key Features

- Pivot table-based analysis  
- Interactive filtering (Region/Product)  
- Revenue and profit tracking  
- Product and region performance comparison  

---

## 💡 Key Insights

- Laptop category drives the highest revenue contribution  
- Desktop category shows higher profitability  
- Certain regions outperform others consistently  
- Performance gap observed between actual and expected values
